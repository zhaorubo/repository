# 公有属性

## 属性的设置与获取

> node.setAttribute('属性','属性值') -- 去设置属性值
> node.getAttribute('属性') -- 获取属性值

```
<body>
        <div class="box haha" id="qwe">我是盒子</div>
        <script>
            let box = document.querySelector('.box')
            console.log(box.classbox) // 不能获取
            console.log(box.getAttribute('class'))
            console.log(box.getAttribute('id'))
            box.setAttribute('class', 'good')
        </script>
    </body>
```

# 私有属性

> 通过 node.属性 的方式去获取或者设置对应的属性值

```
 <body>
        <div class="box haha" id="qwe">我是盒子</div>
        <img src="" alt="" />
        <script>
            let img = document.querySelector('img')
            let a = document.querySelector('a')
            a.href = 'www.baidu.com'
            img.src =
                'https://s.cn.bing.net/th?id=OIP-C.kB-Ovasi0GW67-rmwnAcwAHaEo&w=316&h=197&c=8&rs=1&qlt=90&o=6&dpr=1.56&pid=3.1&rm=2'
        </script>
    </body>
```

## 操作文本框

> 文本框 是通过 node.value 获取文本框的值

```
<input type="text" class="ipt" />
<textarea id="iptarea" cols="30" rows="10"></textarea>
<input type="radio" />
<button>点我</button>

  let ipt = document.querySelector('.ipt')
  let btn = document.querySelector('button')
  let iptarea = document.querySelector('#iptarea')

  btn.addEventListener('click', function () {
      console.log(ipt.value)
      console.log(iptarea.value)
  })
```
