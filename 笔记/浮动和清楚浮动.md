# CSS 布局的三种机制及特点

1. 标准流(普通流)

   > 块级元素独占一行，从上到下顺序排列：行内元素从左到右顺序排列，碰到父元素边界时自动换行

2. 浮动（半脱标）

   > 使盒子漂浮起来，让多个块级元素同行显示

3. 定位（完全脱标）

   > 设定盒子在浏览器中的位置，且不占用文本的位置

```
float:left
```

## 清楚浮动

> 因为浮动的元素脱离标准流，不占据原本的高度，所以需要清除浮动

### 方法(面试题)

```
// 1. 双伪元素法
.clearfix:before, .clearfix:after {
  content: "";
  display: table;
}
.clearfix:after {
  clear:both
  // 清除左右浮动
}
.clearfix {
  zoom : 1;
}
```
